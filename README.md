# NiconiCompare

ニコニコ動画を視聴しながら動画同士を比較し、好みをカテゴリ別のレーティングとランキングとして記録するブラウザ拡張機能です。

データはブラウザ内に保存され、比較履歴の修正、ランキング表示、JSON・CSV・TSV・共有用 HTML の出力に対応しています。

## 開発

```bash
mise install
pnpm install
pnpm dev
```

コミット前に `pnpm fix` と `pnpm check` を実行します。詳細は [開発者ガイド](docs/developer-guide.md) を参照してください。
